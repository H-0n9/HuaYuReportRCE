# HuaYuReportRCE
### 前提

请熟读[中华人民共和国网络安全法](http://mzj.cq.gov.cn/u/cms/cqyfyl/201910/18103505ap94.pdf)，仅限授权安全测试使用,禁止未授权非法攻击站点，一切后果自负

### 使用方法
把资产导入到url.txt，与HuaYuRCE.exe放在一起，执行exe即可
